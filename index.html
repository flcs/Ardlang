About

The Ardlang language is a language specifically tailored for the Arduino. The Arduino is an open-source electronics prototyping platform based on flexible, easy-to-use hardware and software. It's intended for artists, designers, hobbyists, and anyone interested in creating interactive objects or environments. Arduino already comes with its own language, a subset of C++, but a language specifically build for the Arduino has many advantages like:
  • Self-documenting code
  • Program validation
  • Domain specific error messages
  • Easier syntax
  • Less code
  • Domain specific language constructs
Ardlang is developed in the Spoofax Language Workbench and is integrated into the Eclipse IDE.

Example

In this example we set pin 13 as an output pin, and then repeatedly turn it on and off with a 200 millisecond interval.

pin configuration:
output pin1 13

repeat:
set pin1 high
wait 200ms

set pin1 low
wait 200ms
